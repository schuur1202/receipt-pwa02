# レシートCSV変換 PWA（Gemini版）

Google Gemini APIを使ったレシート読み取りアプリです。
**クレジットカード不要・完全無料**で使えます。

---

## APIキーの取得（無料・1分）

1. https://aistudio.google.com/apikey を開く
2. Googleアカウントでログイン
3. 「Create API key」をクリック
4. 表示された `AIza...` で始まる文字列をコピー

---

## GitHub Pages へのデプロイ手順

### 1. リポジトリを作成
1. https://github.com にログイン
2. 右上「+」→「New repository」
3. 名前: `receipt-pwa`、**Public** を選択して作成

### 2. ファイルをアップロード
作成後の画面で「uploading an existing file」をクリックし、
このZIPを解凍したファイルをすべてドラッグ＆ドロップ → 「Commit changes」

### 3. GitHub Pages を有効化
1. リポジトリの「Settings」→「Pages」
2. Source を「GitHub Actions」に変更
3. 1〜2分でデプロイ完了

アクセスURL: `https://ユーザー名.github.io/receipt-pwa/`

---

## スマホへのインストール

### iPhone（Safari）
1. SafariでURLを開く
2. 下の共有ボタン →「ホーム画面に追加」→「追加」

### Android（Chrome）
1. ChromeでURLを開く
2. 「ホーム画面に追加」バナーをタップ、または「⋮」メニューから追加
